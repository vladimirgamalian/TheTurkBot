# The Turk Bot
## The Turk
**The Turk**, also known as the **Mechanical Turk** or **Automaton Chess Player**, was a fake chess-playing machine constructed in the late 18th century ([*wikipedia*](https://en.wikipedia.org/wiki/The_Turk)).
